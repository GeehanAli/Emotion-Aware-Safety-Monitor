# Literature Review

## Key Research Papers

### 1. Leiva-Bianchi et al. (2025)
Systematic review of 40+ grooming detection studies showing:
- SVM achieves best F1-score (0.79)
- MLP achieves highest accuracy (92%)

### 2. Meyer (2015)  
Research on detecting adults pretending to be children using language patterns

### 3. Anderson et al. (IEEE)
Forensic grooming detection system using fuzzy-rough feature selection

### 4. Kim et al. (2025)
Multi-modal grooming detection with 95.3% accuracy using OCR + transformers

## Research Gaps Identified
- Limited work on real-time gaming environments
- Most research uses static chat logs
- Few solutions for live, in-game chat detection
